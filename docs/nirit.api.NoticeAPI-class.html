get(self, object_code, start=0, replies_only=False)

source code 

Provides GET access to Notices and their replies.

Parameters:
  • object_code (string) - The Building Codename or Notice ID.
  • start (number) - The index from which to return results for. By default, the results contains the first N notices or replies, where N is defined in 'nirit.settings.BOARD_NOTICES'.
  • replies_only (boolean) - Specifies whether the result should only contain the replies.
Returns: dict
A dictionary of native Python objects, compatible with the JSON format.

source code 

Provides POST access to editing and sending Notices.

Parameters:
  • subject (string) - The notice or reply subject line.
  • sender (string) - The sender's username.
  • buildings (string) - A coma-separated list of Building codenames. Used as the recipient Building for any new Notice.
  • nid (integer) - When no None, the notice ID to reply to.
  • is_official (boolean) - Whether the post is official or not (1: yes, 0:no)
Returns: dict
An object representing the response status
