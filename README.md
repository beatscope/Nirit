Introduction
------------
Nirit is a business to business network. It aims at connecting local businesses. It is primarily focused on serviced offices, where multiple companies co-habit anonymously.

Requirements
------------
- Python 2.7+
- Django 1.5+
- Django REST framework 2.3+
